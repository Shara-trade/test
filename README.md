## 🚀 Быстрый старт

###1. Установка зависимостей
```bash
pip install -r requirements.txt
```

###2. Настройка
Открой `config.py` и укажи:
```python
BOT_TOKEN = "твой_токен_от_BotFather"
ADMIN_IDS = [твой_telegram_id] # ID админов (можно узнать через @userinfobot)
```

###3. Запуск
```bash
python main.py
```

## 🎮 Игровой процесс

1. Нажми `/start` для регистрации
2. Жми ⛏ **Шахта** и кликай «ДОБЫТЬ АСТЕРОИД»
3. Следи за энергией — она тратится на каждый клик
4. Копи металл и покупай дронов в 🚀 **Ангаре**
5. Дроны добывают ресурсы автоматически
6. Прокачивайся, торгуй, объединяйся в кланы!

## 🛠 Технологический стек

- **Python3.10+**
- **Aiogram3.x** — фреймворк для Telegram
- **SQLite3** — база данных (aiosqlite для async)
- **APScheduler** — планировщик задач

## 🔐 Админ-команды

- `/admin` — панель администратора
- `/find_user ID` — поиск игрока
- `/give ID ресурс количество` — выдача ресурсов
- `/broadcast текст` — рассылка

## 📝 Примечания

- База данных создаётся автоматически при первом запуске
- Все handler'ы асинхронные для высокой производительности
- Система плагинная — легко добавлять новые модули
