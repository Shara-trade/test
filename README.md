# Asteroid Miner Bot

Космический idle-clicker с RPG элементами для Telegram.

## Особенности

- 🚀 **Добыча ресурсов** — кликай по астероидам, собирай металл, кристаллы и тёмную материю
- 🤖 **Система дронов** — пассивный доход от автоматических добытчиков
- ⚙️ **Модули** — генерируемые предметы с бафами и дебафами
- 📦 **Контейнеры** — 5 типов редкости с материалами
- 🔥 **Система перегрева** — риск и награда при активной добыче
- 👥 **Кланы** — корпорации игроков с боссами
- 🏆 **Топы** — соревнование за места в рейтинге

## Установка

### 1. Клонирование

```bash
git clone <repository-url>
cd asteroid-miner
```

### 2. Виртуальное окружение

```bash
python -m venv venv

# Windows
venv\Scripts\activate

# Linux/macOS
source venv/bin/activate
```

### 3. Зависимости

```bash
pip install -r requirements.txt
```

### 4. Конфигурация

Создайте файл `.env` на основе `.env.example`:

```env
BOT_TOKEN=your_bot_token_here
DATABASE_PATH=asteroid_miner.db
ADMIN_IDS=123456789,987654321
```

### 5. Запуск

```bash
python main.py
```

## Игровые механики

### Добыча

- Базовая добыча: 10 металла за клик
- Расход энергии: 10 за клик
- Шанс крита: 3% (x2, x5, x10)
- Шанс контейнера: 1%
- Шанс материала: 5%

### Уровни

Формула опыта: `1000 * level * (1 + 0.1 * level)`

| Уровень | Опыт | Титул |
|---------|------|-------|
| 1 | 1,100 | Новичок |
| 10 | 20,000 | Шахтёр |
| 50 | 300,000 | Галактический барон |
| 100 | 1,100,000 | Легенда космоса |

### Модули

5 редкостей с бафами и дебафами:

| Редкость | Бафов | Дебафов | Шанс |
|----------|-------|---------|------|
| ⚪ Обычная | 1-2 | 2-3 | 70% |
| 🟢 Редкая | 2 | 1-2 | 20% |
| 🟣 Эпическая | 2-3 | 1-2 | 7% |
| 🔴 Мифическая | 3-4 | 0-1 | 2.5% |
| 🟡 Легендарная | 4 | 0-1 | 0.5% |

### Контейнеры

| Тип | Металл | Кристаллы | Шанс |
|-----|--------|-----------|------|
| 📦 Обычный | 50-100 | 10-25 | 65% |
| 🎁 Редкий | 150-300 | 40-80 | 20% |
| 💎 Эпический | 400-800 | 120-250 | 9% |
| 👑 Мифический | 900-1800 | 300-600 | 4% |
| 🔥 Легендарный | 2000-5000 | 800-2000 | 2% |

### Дроны

5 типов с пассивным доходом:

| Тип | Доход/тик | Цена |
|-----|-----------|------|
| Базовый | 2 | 100 металла |
| Шахтёр | 8 | 500 металла |
| Лазерный | 25 | 2,500 металла |
| Квантовый | 70 | 10,000 металла + 50 кристаллов |
| ИИ-дрон | 200 | 50,000 металла + 100 кристаллов + 5 тёмной материи |

## Миграции

```bash
# Проверить статус
python -m database.migrate status

# Применить все
python -m database.migrate apply

# Откатить последнюю
python -m database.migrate rollback
```

## Безопасность

- Токен хранится в `.env` (не коммитится)
- Callback_data подписывается HMAC-SHA256
- Защита от спама (rate limiting)
- Защита от быстрых кликов (ботов)
- Проверка владения ресурсами

## Технологии

- **Python 3.10+**
- **Aiogram 3.x** — Telegram Bot API
- **SQLite** — база данных (aiosqlite)
- **Redis** — кэширование (опционально)
- **APScheduler** — фоновые задачи
